ℹ️ About This App

About the Developer
Mateen Yousuf
Teacher, School Education Department Kashmir
NEP 2020 PARAKH EdTech

This application was designed to help teachers across Jammu & Kashmir and India to move from rote-learning-based assessment to competency-based evaluation, fully aligned with NEP 2020 and NCF 2023 frameworks.

App Features
  • 📖 10 comprehensive policy foundation pages
  • 📐 Blueprint generator with cognitive mapping
  • 📄 Auto question paper from local bank
  • 🧠 Rule-based HOTS question generator
  • 📊 Post-exam analysis & item analysis
  • ✅ Self-evaluation audit tool
  • 🔒 100% offline — no internet needed
  • 💾 Local storage for saving drafts
  • 📱 PWA — installable like an app
  • 🖨️ Print-ready outputs
📱 How to Install as App (PWA)
📱 On Android (Chrome)
  1. Open the app in Chrome
  2. Tap the ⋮ menu (top right)
  3. Select "Add to Home Screen"
  4. Confirm — icon appears on home screen
💻 On Desktop (Chrome/Edge)
  1. Open the app in Chrome or Edge
  2. Click the install icon (📥) in address bar
  3. Click "Install" in the dialog
  4. App launches in its own window
🌐 To Host Free
  1. Create GitHub account (free)
  2. Upload the 4 files to a repo
  3. Enable GitHub Pages in Settings
  4. Share the URL with teachers!
