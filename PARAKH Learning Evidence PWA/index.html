About This App

This app shifts focus from attendance-based tracking to real learning evidence capture, aligned with PARAKH assessment reform, NCF 2023, and NIPUN Bharat Mission. Designed for low-resource school environments across Jammu & Kashmir.

🚀 How to Use / Host
1. Run Locally:
Open index.html with VS Code Live Server or any local server (e.g., npx serve .)

2. Install as App:
Open in Chrome → Menu → "Add to Home Screen"

3. Host Free:
Upload folder to GitHub Pages or Netlify for free hosting
